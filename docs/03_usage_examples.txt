ПРИКЛАДИ ВИКОРИСТАННЯ
=====================

БАЗОВЕ ВИКОРИСТАННЯ
-------------------

Залишити коментарі під постом:
  python main.py --link https://t.me/channel/123 --comments "Супер!" "Клас!" "Дякую!" --count 5

З кастомними параметрами:
  python main.py \
    --link https://t.me/c/1234567890/123 \
    --comments "Чудово" "Цікаво" "Дякую за інформацію" \
    --count 10 \
    --threads 3 \
    --delay 10-20


ФОРМАТИ ПОСИЛАНЬ
----------------

Публічний канал:
  --link https://t.me/channel_name/123
  --link t.me/channel_name/123

Приватний канал / група (закритий по ID):
  --link https://t.me/c/1234567890/123
  --link tg://privatepost?channel=1234567890&post=123

Для приватних каналів/груп, куди акаунти ще не підписані,
потрібно додати інвайт-лінк:
  python main.py \
    --link https://t.me/c/1234567890/123 \
    --invite https://t.me/+INVITE_HASH \
    --comments "Круто!" "Дякую!" \
    --count 10

Пояснення:
  - --link   - посилання на конкретний пост у приватному каналі/групі
  - --invite - інвайт-лінк формату https://t.me/+XXX для автоматичної підписки
  - якщо акаунт вже підписаний, інвайт не потрібен


ФОРМАТ КОМЕНТАРІВ
-----------------

Кілька варіантів (вибираються випадково):
  --comments "Супер!" "Класно!" "Дякую за контент!"

Один коментар:
  --comments "Дуже корисна інформація!"

Коментарі з кількома словами (в лапках):
  --comments "Дуже цікава публікація" "Дякую за корисну інформацію"

Короткі коментарі:
  --comments "+" "like" "top"


ПАРАМЕТРИ КОМАНДНОГО РЯДКА
--------------------------

Обов'язкові:
  --link, -l       Посилання на пост
  --comments, -m   Список коментарів (в лапках)

Опціональні:
  --count, -c      Кількість коментарів (за замовч.: 10)
  --threads, -t    Паралельних потоків (за замовч.: 5)
  --delay, -d      Затримка в секундах (за замовч.: 5-15)
  --invite, -i     Інвайт-лінк для приватних каналів
  --dry-run        Тестовий запуск без надсилання
  --config         Шлях до config.yaml
  --sync           Синхронізація сесій з БД
  --stats          Статистика акаунтів
  --clear-history  Очистити історію для поста


УПРАВЛІННЯ АКАУНТАМИ
--------------------

Синхронізація сесій:
  python main.py --sync

Статистика:
  python main.py --stats

Очистити історію коментарів для поста:
  python main.py --link https://t.me/channel/123 --clear-history


ТЕСТУВАННЯ
----------

Тестовий запуск (без реальних коментарів):
  python main.py \
    --link https://t.me/channel/123 \
    --comments "Тест" "Перевірка" \
    --count 5 \
    --dry-run


ТИПОВІ СЦЕНАРІЇ
---------------

Накрутка коментарів на новий пост:

  1. Синхронізувати акаунти
     python main.py --sync

  2. Перевірити статистику
     python main.py --stats

  3. Залишити коментарі
     python main.py \
       --link https://t.me/my_channel/999 \
       --comments "Круто!" "Дякую!" "Цікаво!" "Супер контент!" \
       --count 20 \
       --threads 3 \
       --delay 10-25

  4. Переглянути результати
     python main.py --stats


Коментування приватного каналу:

  python main.py \
    --link https://t.me/c/1234567890/123 \
    --invite https://t.me/+ABC123xyz \
    --comments "Дякую!" "Корисно!" \
    --count 15


Щоденна робота (bash скрипт):

  #!/bin/bash
  # daily_comments.sh

  # Синхронізація
  python main.py --sync

  # Коментарі на пости
  python main.py --link https://t.me/channel1/100 \
    --comments "Супер!" "Клас!" "Дякую!" --count 10

  python main.py --link https://t.me/channel2/200 \
    --comments "Цікаво!" "Корисно!" --count 15

  # Статистика
  python main.py --stats


ВИРІШЕННЯ ПРОБЛЕМ
-----------------

Помилка "CHANNEL_PRIVATE":
  Акаунти не підписані на приватний канал.
  Рішення:
    - Додайте --invite з інвайт-лінком
    - Або підпишіть акаунти вручну

Помилка "FLOOD_WAIT":
  Telegram обмежує запити.
  Рішення:
    - Збільшити затримку: --delay 20-40
    - Зменшити потоки: --threads 2
    - Почекати та спробувати пізніше

Помилка "CHAT_WRITE_FORBIDDEN":
  Коментарі вимкнені в каналі.
  Рішення:
    - Неможливо коментувати, виберіть інший пост

Помилка "SLOWMODE_WAIT":
  Канал має затримку між повідомленнями.
  Рішення:
    - Збільшити затримку відповідно до slow mode

Помилка "NO_AVAILABLE_ACCOUNTS":
  Всі акаунти вичерпали денний ліміт або вже коментували.
  Рішення:
    - Почекати до наступного дня
    - Додати більше акаунтів
    - Очистити історію: --clear-history
