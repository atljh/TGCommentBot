ПРОКСІ ТА БЕЗПЕКА
=================

ТИПИ ПРОКСІ
-----------

SOCKS5 (рекомендовано):
  socks5://username:password@ip:port
  socks5://ip:port

HTTP/HTTPS:
  http://username:password@ip:port
  http://ip:port


ФОРМАТ PROXIES.TXT
------------------

Приклад файлу proxies.txt:
  socks5://user1:pass1@1.2.3.4:5555
  socks5://user2:pass2@5.6.7.8:5555
  http://user3:pass3@9.10.11.12:8080


РЕКОМЕНДАЦІЇ ПО ПРОКСІ
----------------------

Типи провайдерів:

1. Резидентські проксі (найкраще)
   - Реальні IP від провайдерів
   - Менший ризик блокування
   - Дорожче

2. Мобільні проксі
   - IP від мобільних операторів
   - Високий траст
   - Повільніше, дорожче

3. Датацентрові проксі
   - Дешевші
   - Вищий ризик блокування
   - Підходять для прогрітих акаунтів

Загальні рекомендації:
  - Один проксі = один акаунт
  - Не використовувати безкоштовні проксі
  - Перевіряти геолокацію (близько до номера телефону)
  - Тестувати перед використанням


БЕЗПЕКА АКАУНТІВ
----------------

Рекомендовані ліміти для коментування:

  Параметр                    Значення
  ------------------------------------------
  Коментарів на добу          15-20
  Затримка між коментарями    5-30 сек
  Мін. вік акаунта            7 днів
  Паралельних потоків         3-5

Налаштування в config.yaml:
  safety:
    max_comments_per_day: 20
    delay_range: [5, 15]


ПРОГРІВ АКАУНТІВ
----------------

Нові акаунти потребують прогріву:

Тиждень 1:
  - Лише читання каналів
  - 1-2 коментарі на день
  - Підписка на популярні канали

Тиждень 2:
  - 5-10 коментарів на день
  - Спілкування в чатах
  - Зміна аватарки та імені

Тиждень 3+:
  - Повне використання
  - До 15-20 коментарів


АНТИБАН ПРАВИЛА
---------------

1. Затримки
   - Випадкові затримки між діями
   - Не менше 5 секунд між коментарями

2. Час активності
   - Працювати в "людський" час
   - Не 24/7

3. Різноманітність
   - Різні тексти коментарів
   - Не однакові коментарі від всіх акаунтів

4. Географія
   - Проксі з регіону номера телефону
   - Уникати різких змін локації


ОБРОБКА ПОМИЛОК
---------------

FloodWaitError
  Акаунт отримав flood limit від Telegram.
  Автоматично чекає вказаний час.

SlowModeWaitError
  Канал має slow mode (затримка між повідомленнями).
  Скрипт покаже необхідний час очікування.

UserBannedInChannelError
  Акаунт забанений в каналі.
  Пропускається для цього поста.

ChannelPrivateError
  Акаунт не підписаний на канал.
  Використовуйте --invite для автоматичної підписки.

ChatWriteForbiddenError
  Коментарі вимкнені в каналі.
  Неможливо коментувати цей пост.

SessionRevokedError
  Сесія відкликана (вилогінений).
  Потрібна повторна авторизація.


ПЕРЕМІЩЕННЯ АКАУНТІВ
--------------------

При серйозних помилках акаунти автоматично переміщуються:

  sessions_banned/      - забанені акаунти (BANNED)
  sessions_spam/        - спам-ліміти (SPAM)
  sessions_restricted/  - обмежені акаунти (RESTRICTED)

Це дозволяє:
  - Автоматично виключати проблемні акаунти
  - Зберігати сесії для можливого відновлення
  - Відстежувати статус акаунтів


МОНІТОРИНГ
----------

Статистика акаунтів:
  python main.py --stats

Показує:
  - Статус активності кожного акаунта
  - Кількість коментарів сьогодні
  - Дату останнього використання
